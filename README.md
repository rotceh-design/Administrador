# Facility Management - Sistema de Mantenimiento

Sistema profesional de gestión de mantenimiento para edificios.

## Características

- **Dashboard** con KPIs y gráficos por categoría
- **Tareas** de mantenimiento con filtros por edificio y categoría
- **Cronograma** visual con 4 columnas (Marriott, San 1, San 2, Valparaíso)
- **Carta Gantt** para visualización temporal de actividades
- **Visitas** a edificios con registro completo
- **Incidencias** con prioridad y seguimiento
- **Informes Diarios** de actividades
- **Proveedores** directorio de servicios
- **Fotografías** con compresión automática
- **Correos** generador con 6 plantillas y vista previa
- **Cotizaciones** con cálculo de IVA e impresión PDF
- **Reportes** exportables (CSV/Imprimir)

## Edificios

- Marriott
- San 1
- San 2
- Valparaíso

## Categorías

Electricidad, Gasfitería, Mantenciones, Infraestructura, Climatización, Reparación, Terminaciones, Visitas, Extintores, Vidrios, Jardinería

## Tecnologías

- HTML5, CSS3, JavaScript vanilla
- IndexedDB (almacenamiento local)
- Chart.js (gráficos)
- Font Awesome (iconos)

## Despliegue

El proyecto se despliega automáticamente en Vercel al hacer push a la rama `main`.

## Uso

Abre `index.html` en tu navegador o visita la URL de Vercel.
